# Fine-tuned ALLaM Model

The fine-tuned ALLaM model is hosted on Hugging Face Hub due to its size.

**Model Repository**

https://huggingface.co/Belkacemdz/allam-deepbac-adapter

This repository contains the QLoRA adapter obtained by fine-tuning:

- Base Model: ALLaM-AI/ALLaM-7B-Instruct-preview
- Fine-tuning Method: QLoRA
- Framework: Unsloth + PEFT

To use the model, download the adapter from the Hugging Face repository above.
